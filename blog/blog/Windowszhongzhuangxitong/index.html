Windows重装系统

# 准备

  • 备份数据到其他U盘或其他电脑

  • 一个没有数据的8g以上U盘

  • md5校验工具: https://pan.baidu.com/s/1neMDkyEg0tHlWmwtG2k7BQ 密码:1zi6

  • UltraISO刻录工具: https://cn.ultraiso.net/xiazai.html

  • ISO系统镜像: https://msdn.itellyou.cn/

个人建议重装系统嘛，就是全部重来要清除所有冗余的数据，所以建议把需要的数据都备份到U盘或者其他电脑，然后重装系统的时候整个电脑格式化，装完系统后就很干净了，然后再把备份的数据转进去

# 1.查看处理器

64位处理器可以安装32位和64位操作系统，但是一般安装64位系统，性能更好

32位处理器只能安装32位操作系统

查看电脑处理器是否支持64，还是只能32位：在我的电脑右键点击属性

  • 如果此处为64位，那么处理器便支持64位操作系统
  • 如果为32位，不一定就只能支持32位操作系统，有可能之前安装了32位的系统, 点击查看是否支持64位

# 2. 下载系统

点击MSDN下载需要的系统

下载HashMyFiles(链接在文章开头)，将下载好的hashmyfiles解压

打开hashmyfiles检验md5是否正确

将下载的ISO镜像拉入HashMyFiles窗口

等待进度走完对比SHA1是否正确

# 3. 制作系统盘

点击下载刻录工具

打开UltraISO，点击继续试用

  1. 选择文件->打开, 打开下载好的系统ISO文件

  2. 插入准备好的U盘

  3. 选择启动 -> 写入硬盘映像

点击写入等进度条完成就制作完成了

# 4. 如何进入U盘启动

不同品牌电脑不一样，有些需要进入BIOS设置，有些直接可以开机选择启动

自行百度: 比如"联想u盘启动"

我的古董联想开机出现开机画面狂戳F12就可以选择U盘启动了

有时候电脑开机太快来不及反应按键，可能是打开了快速启动, 把它关了

相应的，安装完系统可以到这里把它打开有助于快速开机

# 5. 开始重装系统

进入U盘启动后

# 5.1 普通设置

直接跳过，安装完再弄

这里选择需要的版本，我选择了专业版

下面的我选择自定义然后将服务都取消了进入下一步

# 5.2 硬盘分区

  • 不需要重新分区，分别选中驱动器0每个分区然后点击格式化
  • 需要重新分区，把我的电脑硬盘即驱动器0每个分区分别选中并点击删除，然后重新分区

选中驱动器0，然后点击新建依次建立分区

我的建议是：系统盘50g，软件盘30g(如果硬盘比较大就50g以上)，其他一个盘放文件

分区完成点击下一步

# 5.3 最后设置

完成以上后等一会儿进入下面画面进入最后设置

后面的设置自己设置好经过一两次重启就大功告成了来到了主界面

以上就打开了桌面的图标了，然后自己通读系统每一个设置选项设置好电脑
